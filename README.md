# map-memory

一个纯前端的地图记忆训练网页项目。

当前目标不是做地图浏览器或百科站，而是把地图区划学习做成真正的主动回忆训练，先打透两条主线：

- 世界上每一个国家
- 中国每一个地级行政区

## Stack

- Vite
- React
- TypeScript
- Bun

## Run

```bash
bun install
bun dev
```

## Build

```bash
bun run build
```

## Project Intent

这个项目要解决的核心问题不是“用户能不能认出一张地图”，而是“用户能不能通过反复训练，
真正把地图区划记在脑子里”。

因此后面的产品设计会优先围绕这些原则展开：

- 以主动回忆为中心
- 把空间位置和名称绑定起来
- 把世界国家和中国地级行政区放进同一个训练框架
- 尽早把错题回放做进去，而不是只做一次性 quiz
